This folder is intended to store public landing-page images for the BhoomiChain portal.

Expected files (you can drop your own PNG/JPG assets with these exact names):
- hero-land-map.png              (hero land parcel / GIS map visual)
- land-registry-workflow.png     (simple workflow diagram for land registry)
- land-dashboard.png             (screenshot or mock of the BhoomiChain dashboard)
- land-plots-aerial.jpg          (aerial photograph of land plots / cadastral layout)

Until you provide real images, the landing page will reference these paths but the browser may show broken-image placeholders.
